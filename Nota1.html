Qu es EDA?

Anlisis Exploratorio de Datos, o Exploratory Data Analysis (EDA), es un proceso que consiste en explorar y comprender los datos. El EDA es fundamental en cualquier anlisis de datos, ya que ayuda a comprender la calidad de los datos y a identificar reas productivas para investigar a profundidad.

En el que:

  1. Generamos preguntas sobre los datos.
  2. Buscamos respuestas visualizando, transformando y modelando los datos.
  3. Utilizamos lo que aprendimos para refinar las preguntas y/o generar nuevas preguntas.

Acerca de las preguntas

Tipos de Variables

Tipo de Variable Descripcin
Cualitativa Nominal Representa categoras sin orden inherente.
Cualitativa Ordinal Representa categoras con un orden especfico.
Cuantitativa Continua Representa valores numricos con infinitos puntos entre ellos.
Cuantitativa Discreta Representa valores numricos con saltos finitos entre ellos.

Cmo se presenta los datos?

  • Variable: Es una cantidad, cualidad o propiedad que se puede medir.

  • Valor: Es el estado de una variable cuando se mide. El valor de una variable puede cambiar de medicin a medicin.

  • Observacin o Caso: Es un conjunto de mediciones realizadas bajo condiciones similares (generalmente todas las mediciones en una observacin se realizan al mismo tiempo y en el mismo objeto).

  • Datos Tabulares: Es un conjunto de valores, cada uno asociado con una variable y una observacin. Los datos tabulares estn ordenados si cada valor se coloca en su propia celda, cada variable en su propia columna y cada observacin en su propia fila.

T_freq <-  diamonds %>%  # Seleccionamos la base de datos 
  count(cut) %>% # Contamos obs por categoria cut
  rename(Fi = n) %>% 
  mutate(Pi = round(Fi/sum(Fi), 2), Po = Pi * 100) # Agregamos las columnas
